O projeto "envia" está configurado no springboot pra rodar na porta 8081
O projeto "recebe" está configurado no springboot pra rodar na porta 8082


o link do github onde está o projeto é:

              https://github.com/evertonidpb/projeto_ActiveMQ


Para acessar a documentação "Envia" copie a pasta dist para dentro do tomcat e  acesse:
                           http://localhost:8080/dist/envia.html                 

Para acessar a documentação "Recebe" acesse:
                           http://localhost:8080/dist/recebe.html
